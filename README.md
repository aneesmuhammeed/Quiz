# MCQ Exam Platform

Serverless MCQ exam platform built with React and Supabase.

## Setup

1. Install dependencies:
   npm install
2. Create a .env file in the project root:
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key

## Development

npm run dev

## Supabase

Run the SQL in supabase.sql to create the submissions table and RLS policy.

## Notes

Replace the placeholder questions in src/App.jsx with official exam questions before production use.
